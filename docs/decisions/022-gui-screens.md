# ADR-022: GUI 화면 구성

## 상태

`결정됨`

## 날짜

2026-03-25

## 맥락

GUI에 어떤 화면과 정보를 표시할지 결정해야 한다.

## 결정

### 실시간 업데이트

WebSocket으로 브라우저에 푸시하는 데이터:

```javascript
// 1초마다: 포지션 현재가, 미실현 P&L, 보유 시간
{ type: "position_update", data: { symbol, price, pnl, elapsed_min } }

// 이벤트 발생 시: 포지션 진입/청산, 스캐너 선정, 경고
{ type: "event", data: { event_type, message, timestamp } }

// 1분마다: 자산 현황, 일일 통계
{ type: "stats_update", data: { balance, daily_pnl, trade_count, ... } }
```

### 차트 라이브러리

- **수익률 차트:** Lightweight Charts (TradingView 오픈소스) — 금융 차트에 최적화
- **통계 차트:** Chart.js — 바 차트, 파이 차트 등
- 서버에서 데이터만 JSON으로 제공, 렌더링은 브라우저에서

